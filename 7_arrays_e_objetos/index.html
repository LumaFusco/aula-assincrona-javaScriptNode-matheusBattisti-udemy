INTRODUÇÃO

PARTE 1

Arrays

  • Permite adicional um conjunto de valores a uma variável
  • Deve ser escrito entre colchetes, separando os valores por vírgula. Exemplo:
    let numbers = [1, 2, 4, 81];

Mais sobre Arrays

  • Podemos acessar o array pelo seu indice. Lembrando que o primero é sempre 0. Ou seja, para acessar o primeiro elemento: arr[0]

PARTE 2

Propriedades

  • Propriedades são informações que podem ser verificadas de um valor.
  • Elas aparacem com "nomeDoArray.algumacoisa" ou ainda "nomeDoArray['algumacoisa']
  • Com excessão a NULL e UNDEFINED, todos os valores dem JavaScript tem propriedades.
  • PROPRIEDADE LENGTH: mostra quantos valores há no array.

PARTE 3

Métodos

  • Métodos são propriedade que funcionam como FUNÇÕES.
  • Strings e Arrays contém métodos!
  • Ao serem invocadas, necessitam de (). Exemplo:
    console.log(nomeDoLet.toUpperCase());

    PARTE 11

    Métodos de Array: PUSH e POP

  • Pop = retira | Push = inclui.
  • Funcionam sempre no FINAL do array (último elemento)

    • PARTE 12

    Métodos de Array: SHIFT e UNSHIFT

  • Funcionam sempre no COMEÇO do array
  • Shift: remove o primeiro
  • Unshift: adiciona um ou mais elementos no começo do array

    • PARTE 13

    Métodos de Array: indexOf e lastIndexOf

  • indexOf: encontra o indice (posição) de um determinado elemento.
  • lastIndexOf: entra o último índice de um elemento
  • Lembrando que a lista sempre começa com 0

    • PARTE 14

    Método de Array: SLICE

  • Retorna um array a partir de outro array
  • O array retornado é determinado pelos parâmetros que são os índices de início e fim do novo array
  • O indice sempre para um na frente!
  • Pode ser feito de trás para frente, usando o sinal de menos

    • PARTE 15

    Método de Array: forEach

  • Itera cada elemento do array. É como se fosse um for ou while mas é um método pronto do JavaScript.
  • Você declara a variavel no plural e quando a chama no singular o JS entende que o singular é cada um doa elementos. Na verdade pode ser qualquer nome, até um i ele entende o que é.

    PARTE 26

    O que é Json?

    Uma padronização de tipo de dados. Não tem MÉTODOS, apenas dados e valores. ATENÇÂO: apenas aspas duplas e não aceita comentários! Ex:

    let pessoa = {
    "name": "Matheus",
    "age": 29,
    "languages": ["PHP, "JavaScript", "Python"]
    }
